⚡ DÉMARRAGE ULTRA-RAPIDE

1. Créez un projet GRATUIT sur supabase.com
2. Récupérez votre URL et anon key
3. Modifiez le fichier .env avec vos clés
4. Exécutez database-setup.sql dans Supabase SQL Editor
5. Lancez : npm install && npm run dev
6. Créez votre compte admin (voir DEMARRAGE-RAPIDE.md)
